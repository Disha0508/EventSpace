# EventSpace

**Система управления мероприятиями / платформа для создания и регистрации на события**

---

## 📌 Описание проекта

**EventSpace** — это веб-приложение для организации мероприятий, регистрации участников, просмотра афиши и управления событиями.

Проект разрабатывается в рамках **учебного курса по ИТ и веб-программированию в БГУИР**.

---

## 🚀 Основной функционал (планируется)

- Создание и управление мероприятиями  
- Регистрация пользователей на события  
- Просмотр списка и афиши мероприятий  
- Управление участниками  

---
